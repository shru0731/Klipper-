# Raspberry Pi Setup

## Overview

This guide explains how to prepare a Raspberry Pi for running Klipper.

---

## Requirements

- Raspberry Pi (3/4/5 or Zero 2 W)
- MicroSD Card (16GB or larger)
- Card Reader
- Raspberry Pi OS Image
- Raspberry Pi Imager
- Power Supply

---

## Step 1: Download Raspberry Pi Imager

Download Raspberry Pi Imager from:

https://www.raspberrypi.com/software/

Install it on your computer.

---

## Step 2: Insert the SD Card

Insert the microSD card into your computer.

---

## Step 3: Flash Raspberry Pi OS

Open Raspberry Pi Imager.

Choose

Operating System
→ Raspberry Pi OS Lite (64-bit)

Choose your SD Card.

Click **Next**.

---

## Step 4: Configure Settings

Before writing:

- Set Hostname
- Enable SSH
- Configure Wi-Fi
- Set Username
- Set Password
- Set Timezone

Click Save.

---

## Step 5: Flash the SD Card

Click Write.

Wait until flashing is complete.

---

## Step 6: Boot Raspberry Pi

Insert the SD card into the Raspberry Pi.

Power it on.

The Raspberry Pi is now ready.
