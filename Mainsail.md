# Install Mainsail

## Launch KIAUH

```bash
git clone git clone https://github.com/dw-0/kiauh.git
cd ~/kiauh
./kiauh.sh
```

---

## Install

Choose

```
Install
```

Select

```
Mainsail
```

---

## Open Browser

Find Raspberry Pi IP

```bash
hostname -I
```

Example

```
192.168.0.120
```

Open

```
http://192.168.0.120
```

The Mainsail dashboard should appear.

---

## Verify

You should see

- Printer Status
- Console
- Temperature Graph
- Files

Mainsail installation is complete.
